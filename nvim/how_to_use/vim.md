# Tabの使い方
## Tab移動
gt 次のタブへ
gT 前のタブへ
{i}gt iのタブへ

## Tabの順番移動
:tabmove {i} 現在のタブをi番目に移動
:+tabmove 現在のタブを右に移動
:-tabmove 現在のタブを左に移動

## Tabを一括で閉じる
:tabo 現在のタブ以外のタブをすべて閉じる

## すべてのバッファをtabで開く
:tab ba

# バッファの使い方
## バッファの移動
:b {i} i番目のバッファに移動
:b n   次のバッファに移動
:b p   前のバッファに移動
:b file_name そのファイルのバッファに移動

## バッファ一覧
:ls バッファ一覧を表示

# 補完について
## 補完候補を選択
C-N 下の候補へ
C-P 上の候補へ
C-Y 補完を確定
C-E 補完を中止

## C-X補完
C-X C-F  ファイルパス
C-X C
